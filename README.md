# 🚀 External Vault

## Fong's Adblock List

一款高效、全面的广告过滤规则列表，专为提升网络浏览体验而设计。本项目自动聚合多个高质量过滤源，提供最新、最精简的广告拦截规则。

### 主要特点

- **全面覆盖**：整合多个主流过滤器的精华规则
- **自动更新**：每日 UTC 时间 16:00 自动构建最新规则
- **智能去重**：自动合并多源规则，消除重复项，保持列表精简高效
- **标准格式**：完全兼容 Adblock Plus 2.0 规范，适用于主流广告拦截工具

### 数据来源

本列表聚合以下高质量过滤源：

#### 基础广告过滤

- AdGuard 基础过滤器
- AdGuard 中文特别版

#### 烦人元素过滤

- cjxlist 烦人元素过滤
- Fanboy 烦人元素过滤
- uBlock Origin 烦人元素规则

#### Cookie 专项过滤

- EasyList Cookie 规则
- uBlock Origin Cookie 烦人规则

### 使用方法

1. 在您的广告拦截工具（如 uBlock Origin、AdGuard 等）中
2. 添加自定义过滤列表
3. 输入订阅地址：`https://ex.135468.xyz/adblocklist.txt`
